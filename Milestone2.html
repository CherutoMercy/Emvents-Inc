Milestone2.html Milestone: Build a basic Node.js app with Express.js First things first. We need to get the tooling ready so you can take full advantage of Node.js and its libraries to build the API for Emvents Inc. You will build this Node.js API using Express.js, ES6 features and testing that actually works. Get ready (5 / 5) Complete these tasks by yourself before meeting with your partner. Task #1 Check new JavaScript features (ES6), at least arrow functions and let/const. http://exploringjs.com/es6/ Task #2 Install Node.js in your computer. https://nodejs.org/en/download/ Task #3 Install Postman. https://www.getpostman.com/ Task #4 Learn about git. You should git commit at least after completing each requirement. https://git-scm.com/docs/ Task #5 Review the basic HTTP methods. https://en.wikipedia.org/wiki/Hypertext_Transfer_Protocol#Request_methods Requirements (9 / 9) Complete these requirements during the session with your partner. Requirement #1 Create a Github repo for this project. You and your coding partner can collaborate using this repo. Requirement #2 Every time you complete a requirement, you should push a commit to the repo of the project. Requirement #3 You should start by creating and running a simple Hello World from your terminal. https://nodejs.org/api/synopsis.html Requirement #4 Check the Hello World results in your browser. Requirement #5 You should use npm to initialize a new project Node project. Provide a description and see the resulting package.json file. https://docs.npmjs.com/ Requirement #6 Use npm to install Express.js. http://expressjs.com/en/starter/installing.html Requirement #7 Add Express.js to your project and add a GET and a POST handler to send and receive data from the client. http://expressjs.com/en/starter/hello-world.html Requirement #8 Use Postman to test your basic service works. Requirement #9 Refactor your Express.js code to use ES6 features (const / let and arrow functions when convenient)
