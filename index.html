Build a Backend with Node.js

In this course you will learn how to build a production-ready REST API using Node.js and Express.js.

You will work in a real-world project for a fictional company. The project is in different milestones and has added functional and technical requirements that you will need to implement. Bellow is a step by step guide:

Milestone 1: Introduction
Milestone 2: Build a basic Node.js app with Express.js
Milestone 3: Build an in-memory RESTful API
Milestone 4: Testing your API
Milestone 5: Persisting the API data with Mongodb
Milestone 6: Users and authentication
Milestone 7: Authorization
Milestone 8: Deploy your backend
