Your courses Build a Backend with Node.js In this course you will learn how to build a production-ready REST API using Node.js and Express.js. You will work in a real-world project for a fictional company that requires your help. We have split the project into different milestones and we have added functional and technical requirements that you will need to implement with your coding partner. Are you ready? Milestone 1: Introduction Milestone 2: Build a basic Node.js app with Express.js Milestone 3: Build an in-memory RESTful API Milestone 4: Testing your API Milestone 5: Persisting the API data with Mongodb Milestone 6: Users and authentication Milestone 7: Authorization Milestone 8: Deploy your backend
